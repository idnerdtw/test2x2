🚀 未來開發計劃

Test2x2 專注於 2×2 列聯表分析。原始 aStat v0.6 的其他統計功能，我們計劃開發為獨立的專門應用程式：

📊

人年計算器

人年統計推論，比較暴露與非暴露組別的發生率是否有差異。計算發生率比值和發生率差異。

💊

NNT/NNH 計算器

治療所需人數/傷害所需人數計算，使用 Newcombe 方法提供信賴區間。從事件發生率計算絕對風險減少/增加。

📈

比例信賴區間計算器

使用三種方法計算單一比例的信賴區間：精確方法、Wilson's score 方法、Agresti-Coull 方法。

每個應用程式都將專門化、專注且經過充分測試 - 遵循「專精一事」的哲學。

🚀 Future Development Plans

Test2x2 focuses solely on 2×2 contingency table analysis. The original aStat v0.6 had additional statistical features that we plan to develop as separate specialized apps:

📊

PersonTime Calculator

Person-time statistical inference comparing incidence rates between exposed vs. non-exposed groups. Calculate incidence rate ratios and rate differences.

💊

NNT/NNH Calculator

Number Needed to Treat/Harm calculations with confidence intervals using Newcombe's method. Calculate absolute risk reduction/increase from event rates.

📈

Proportion CI Calculator

Single proportion confidence intervals using three methods: Exact method, Wilson's score method, and Agresti-Coull method.

Each app will be specialized, focused, and thoroughly tested - following the "do one thing well" philosophy.

📖 開發故事

Test2x2 有一個橫跨十多年的有趣開發歷程：

2010

📱 aStat v0.6 - 起源

最初以「aStat version 0.6」開發，當時智慧型手機剛剛興起。HTC Desire 紅極一時，Android API level 4 還是最先進的技術。aStat 包含完整的統計功能：2×2列聯表、人年分析、NNT/NNH計算，以及比例信賴區間。

2010-2024

⏰ 沉寂歲月

aStat 因缺乏維護而逐漸過時。許多細節規範未能完成，最終導致從 Google Play Store 下架。這套完整的統計工具組沉寂了十多年。

2024

🤖 AI 驅動的復興

利用強大的 LLM 工具（包括 Claude AI）重新啟動專案。先在 a0.dev 平台開發 React 原型，再轉換為 Kotlin 原生 Android 應用程式。不過這次我們決定先專注於完善一個核心功能。

2025

🎯 Test2x2 - 專注卓越

發現「aStat」已被多家統計公司使用。決定重新命名並專注於僅做 2×2 列聯表分析。原本 aStat 的其他功能（人年分析、NNT/NNH、比例信賴區間）計劃開發為獨立的專門應用程式。

希望大家喜歡這個經典統計工具的現代復活版本！🎉

📖 Development Story

Test2x2 has an interesting development journey that spans over a decade:

2010

📱 aStat v0.6 - The Beginning

Originally developed as "aStat version 0.6" when smartphones were just emerging. The HTC Desire was all the rage, and Android API level 4 was cutting-edge technology. aStat included comprehensive statistical features: 2×2 contingency tables, person-time analysis, NNT/NNH calculations, and confidence intervals for proportions.

2010-2024

⏰ Years of Dormancy

aStat gradually became outdated due to lack of maintenance. Many detailed specifications were left incomplete, eventually leading to removal from Google Play Store. The comprehensive statistical toolkit remained dormant for over a decade.

2024

🤖 AI-Powered Renaissance

Using powerful LLM tools (including Claude AI), the project was revived. Started as a React prototype on a0.dev platform, then transformed into a native Android app with Kotlin. However, this time we focused on perfecting one core feature first.

2025

🎯 Test2x2 - Focused Excellence

Discovered that "aStat" was already used by multiple statistical companies. Decided to rename and focus on 2×2 contingency table analysis only. The other original aStat features (person-time analysis, NNT/NNH, proportion confidence intervals) are planned as separate specialized apps.

We hope you enjoy this modern revival of a classic statistical tool! 🎉
